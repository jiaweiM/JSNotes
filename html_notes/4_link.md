# 链接

## 简介

`<a>` 标签可用于添加超链接。通过 `href` 属性指定地址：

```html
<a href=""></a>
```

对图片添加超链接，将 `<img>` 嵌入到 `<a>` 中即可。

## target

`target` 属性用于指定在哪打开链接文本。使用 `_blank` 表示在 new window 或 new tab 中打开链接。如：

```html
<a href="http://www.sololearn.com" target="_blank">
   Learn Playing
</a>
```

点开链接后，该链接变为紫色，添加了下划线。
